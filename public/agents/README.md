# Dossier des photos d'agents

Placez ici les photos de tous les agents avec le nom exact correspondant.

## Format de nommage

Les fichiers doivent être nommés **exactement** comme le nom de l'agent avec l'extension `.png`

### Exemples :
- `Deimos.png`
- `Brava.png`
- `Sentry.png`
- `Nøkk.png` (avec le caractère spécial ø)
- `Capitão.png` (avec l'accent)
- `Skopós.png` (avec l'accent)
- `Tubarão.png` (avec le tilde)
- `Jäger.png` (avec le tréma)
